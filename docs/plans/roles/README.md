# 角色提示詞索引

本目錄收錄網站專案各角色的 AI 系統提示詞，可直接複製給 AI agent 使用，讓其扮演對應角色協助工作。角色定義來源見 [`../專案人員配置表.md`](../專案人員配置表.md)。

|---|------|-----------|-----------|
| 1 | 專案經理 PM | [專案經理.md](專案經理.md) | 掌控時程、預算與跨部門溝通，確保如期如質交付 |
| 2 | UI/UX 設計師 | [UIUX設計師.md](UIUX設計師.md) | 規劃使用者動線與視覺設計，讓網站好用又好看 |

## 提示詞結構

每份檔案採一致格式，方便套用到 AI agent：

- **角色設定** — 資歷與專業定位
- **核心職責** — 該角色負責的工作範圍
- **工作準則** — 判斷原則與規範
- **輸出格式** — 語言、術語與結構化要求
- **常見任務** — 可直接指派的典型交付物

## 使用方式

1. 開啟對應角色的 `.md` 檔，複製全文作為 AI 的 system prompt。
2. 接著描述你的具體需求（例如：「為 Contact 頁設計線框圖」）。
3. AI 即會以該角色的專業視角回應。

> 前端、後端、SEO 提示詞已對齊本專案技術脈絡（React 19 + TypeScript + Tailwind v4 + shadcn/ui，部署於 GitHub Pages）。
